# INX Future Inc. Employee Performance & Attrition Analysis

## 📌 Project Overview
This repository contains analysis of the **INX Future Inc. Employee dataset**, which includes **1,200 records** and approximately **28 features**.  
The dataset captures diverse employee attributes such as demographics, career progression, compensation, satisfaction levels, and performance ratings.

The primary objectives of this project are:
- 🔍 **Analyze workforce patterns** across departments, roles, and demographics  
- 📊 **Predict employee performance ratings** using machine learning models  
- 📉 **Identify attrition risks** and key drivers of employee turnover  
- 💡 **Provide HR recommendations** to improve retention and productivity  

---

## 📂 Dataset Features
The dataset includes:
- **Quantitative (numeric)**: Age, Monthly Income, Years at Company, etc.  
- **Ordinal categorical**: Job Level, Performance Rating, Education Level  
- **Qualitative categorical**: Department, Gender, Business Travel, OverTime  

---

## ⚙️ Methodology
1. **Exploratory Data Analysis (EDA)** – Univariate, bivariate, and correlation insights  
2. **Data Preprocessing** – Handling missing values, outliers, and class imbalance (SMOTE)  
3. **Model Building** – Random Forest, SVC, MLP, and other ML algorithms  
4. **Hyperparameter Tuning** – GridSearchCV and RandomizedSearchCV optimization  
5. **HR Insights & Recommendations** – Department‑wise performance and attrition drivers  

---

## 📈 Key Insights
- Department‑wise performance variations highlight areas for targeted HR policies  
- Career progression features (Job Level, Years Since Last Promotion) strongly influence ratings  
- Work‑life balance and overtime are critical predictors of attrition  
- Compensation and satisfaction metrics correlate with retention outcomes
